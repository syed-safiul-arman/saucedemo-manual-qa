# SauceDemo Manual QA Test

## Overview
Manual QA testing of SauceDemo, a public demo e-commerce app maintained by Sauce Labs.

## Project Structure
- /test-plan: Overall test strategy
- /test-cases: Detailed test cases by feature
- /bug-reports: Defects found during testing
- /test-execution: Test run results

## Tools Used
- Browser: Chrome (latest)
- Test Documentation: Markdown
- Bug Tracking: GitHub Issues

## Test Coverage
- Login flow
- Product browsing and sorting
- Cart management
- Checkout flow
- Logout
